# Introduction
Travelcations, a travel application, allowing users to plan their trips proficiently, by providing relevant information, suggesting itineraries for destinations around the world. Also helping users manage time, attractions, budget, and expenses during the journey
# Techstack
We will be developing in native mode with Swift for iOS and Java for Android. Firebase services will be used to authenticate and push notifications. We will also use Realm by MongoDB to store the data locally. We will use Here technologies RestAPIs, the World's Number One Location Platform, for the application's heart and brain, such as fetching the intermodal route, checking for weather, traffic, and fetching the activities.
